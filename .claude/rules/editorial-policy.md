# 編集方針（Editorial Policy）

## 正規定義ファイル
- 禁止表現リスト: `config/compliance.yaml` の `ng_words`
- コンプライアンスチェック実装: `src/generator/compliance.ts`
- 本ルールはこれらのファイルを正とし、補足・運用ガイドラインを定める

## 禁止表現

### NG単語（compliance.yaml定義）
「必ず儲かる」「元本保証」「リスクなし」「確実に稼げる」「損しない」「絶対に利益」「ノーリスク」「guaranteed profit」「risk free」

### 金融クレームパターン（compliance.ts定義）
- `(絶対|確実)(に)?(稼げる|儲かる|利益)`
- `(簡単|誰でも).{0,8}(稼げる|儲かる)`
- `(ノーリスク|risk free|guaranteed profit)`
- `(元本).{0,6}(保証)`

これらが記事中に含まれると自動でブロックされる。

## ヘッジ表現禁止
伝聞・推測の曖昧表現は使わない。事実ベースで書く。

| NG | OK |
|----|-----|
| 〜とされています | 〜と表示されています |
| 〜の可能性があります | 〜が用意されています |
| 〜と言われています | 公式サイトによると〜 |
| 〜かもしれません | （不確実なら書かない） |

## 投資助言回避

### 金融トピックマーカー（compliance.ts定義）
「稼ぎ方」「報酬」「換金」「口座開設」「アフィリエイト」が含まれる記事は自動で手動レビュー対象になる。

### 書き方の原則
- 報酬の説明は「ゲーム内でこう表示されている」という事実の報告に徹する
- 将来の価格・利益を示唆しない
- 換金方法の詳細は書かず、換金ガイド（`/guide/how-to-cash-out/`）へリンクする

## 事実 vs 推定の明示

| 区分 | 書き方 | 例 |
|------|--------|-----|
| 確認済み事実 | 断定形 | 「BOMBトークンはKaiaチェーン上のERC-20トークンです」 |
| ゲーム画面の情報 | 「〜と表示されています」 | 「リワード画面に"100 BOMB"と表示されています」 |
| 推定（`likely`） | 明示的に注記 | 「KaiaScanの記録からBOMBトークンが報酬として使われている可能性が高いですが、公式の確認は取れていません」 |
| 未調査 | 記事に含めない | — |

## 専門用語の初出定義
以下の用語は記事中の**初出時**に1文で定義する:

| 用語 | 定義例 |
|------|--------|
| KAIA | LINE Dapp Portalで獲得できる暗号資産。国内取引所で日本円に換金可能。 |
| NFT | ブロックチェーンに記録された固有のデジタルアイテム。マーケットプレイスで売買可能。 |
| Dapp Portal | LINE上で遊べるブロックチェーンゲーム（Mini Dapp）のプラットフォーム。 |
| Mini Dapp | Dapp Portal上で動作する軽量なブロックチェーンアプリ。 |

## 免責事項
記事末尾に免責事項は**記事本文に含めない**（テンプレートで別途追加）。
テンプレートは `config/compliance.yaml` の `disclaimer_template` に定義済み。
